# cc-prog

ComputerCraft programs.

## Easy Quarry Install

Run this on a ComputerCraft turtle:

```lua
wget run https://raw.githubusercontent.com/raf181/cc-prog/main/mining/quarry/setup.lua
```

Then run:

```lua
quarry
```

## Programs

- `mining/quarry`: resilient quarry turtle program with local tests.
